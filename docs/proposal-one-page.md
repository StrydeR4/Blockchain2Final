# One-Page Proposal

## Scenario

Option B - GameFi Economy.

## Product

The team will build a DAO-governed crafting RPG economy where players collect ERC1155 resources, swap fungible resources through a custom constant-product AMM, craft equipment, rent rare NFT items, and open verifiable random loot drops powered by Chainlink VRF.

## Technical Scope

The protocol includes an ERC20Votes governance token, ERC1155 item system, ERC4626 rental collateral vault, from-scratch AMM, Chainlink price stale-check oracle, Chainlink VRF loot drops, OpenZeppelin Governor and Timelock, UUPS upgradeable config, CREATE/CREATE2 factory, The Graph subgraph, React dApp, L2 deployment scripts, CI, audit report, architecture document, and gas report.

## L2 Target

Base Sepolia, with Arbitrum Sepolia and Optimism Sepolia as alternatives.
